# 门店商品经营 AI 分析助手新手使用教程

这份教程适合第一次接触命令行的新手，按步骤操作即可启动本项目。

## 一、这个项目是做什么的

本项目用于录入门店商品经营数据，例如 SKU 结构、进销存表现、品类结构等，然后调用火山方舟等 OpenAI 兼容大模型，生成门店商品经营分析报告。

启动成功后，你会在浏览器里看到一个网页表单。填写数据后点击“生成分析报告”，系统会输出“易物提销”和“悦人增黏”两个维度的经营建议。

## 二、运行前需要准备什么

你需要准备：

1. 一台 Windows 电脑。
2. 已安装 Python 3.8 或更高版本。
3. 火山方舟 API Key。
4. 火山方舟推理接入点 ID，通常是 `ep-` 开头。
5. 本项目文件夹：`retail-ai-analyzer`。

重要提醒：

- 不要把 API Key 发给别人。
- 不要把 API Key 写进 `main.py` 或上传到公开仓库。
- 建议用环境变量方式配置 API Key。
- 火山方舟的 `AI_MODEL` 建议填写推理接入点 ID，也就是 `ep-xxxxxxxx`，不是普通模型名称。

## 三、确认 Python 是否安装成功

1. 按键盘上的 `Win + R`。
2. 输入：

```cmd
cmd
```

3. 按回车，打开命令提示符。
4. 在 CMD 里输入：

```cmd
python --version
```

如果看到类似下面的内容，说明 Python 已安装：

```text
Python 3.11.6
```

如果提示 `python 不是内部或外部命令`，说明 Python 没有安装，或者安装时没有勾选 `Add Python to PATH`。

可以去 Python 官网下载安装：

```text
https://www.python.org/downloads/
```

安装时请勾选：

```text
Add python.exe to PATH
```

安装完成后，重新打开 CMD，再运行：

```cmd
python --version
```

## 四、用 CMD 进入项目文件夹

假设你的项目在：

```text
C:\Users\你的用户名\Downloads\retail-ai-analyzer
```

在 CMD 中输入：

```cmd
cd /d C:\Users\你的用户名\Downloads\retail-ai-analyzer
```

如果你的项目路径不同，请把后面的路径改成你自己的项目文件夹路径。

例如，如果项目在桌面：

```cmd
cd /d C:\Users\你的用户名\Desktop\retail-ai-analyzer
```

进入成功后，可以输入：

```cmd
dir
```

如果能看到这些文件，说明位置正确：

```text
main.py
requirements.txt
start.bat
README.md
templates
static
```

## 五、创建虚拟环境

虚拟环境可以把这个项目需要的依赖单独放起来，避免影响电脑上的其他 Python 项目。

在项目目录下运行：

```cmd
python -m venv venv
```

运行完成后，项目文件夹里会多出一个 `venv` 文件夹。

## 六、激活虚拟环境

继续在 CMD 中运行：

```cmd
venv\Scripts\activate
```

激活成功后，命令行前面会出现：

```text
(venv)
```

例如：

```text
(venv) C:\Users\你的用户名\Downloads\retail-ai-analyzer>
```

## 七、安装项目依赖

激活虚拟环境后，运行：

```cmd
pip install -r requirements.txt
```

这个命令会安装项目需要的组件，例如：

- FastAPI
- uvicorn
- httpx
- jinja2

如果安装速度很慢，可以尝试使用国内镜像：

```cmd
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 八、配置火山方舟 API

本项目推荐用环境变量配置 API。

在 CMD 中运行以下命令。

第一步，配置 API Key：

```cmd
set AI_API_KEY=你的火山方舟APIKey
```

第二步，配置火山方舟接口地址：

```cmd
set AI_BASE_URL=https://ark.cn-beijing.volces.com/api/v3
```

第三步，配置推理接入点 ID：

```cmd
set AI_MODEL=你的推理接入点ID
```

示例格式：

```cmd
set AI_MODEL=ep-202xxxxxxxxxxxxx
```

注意：

- `AI_API_KEY` 填你的真实 API Key。
- `AI_MODEL` 填火山方舟控制台里的推理接入点 ID。
- 这三个 `set` 命令只在当前 CMD 窗口有效。关闭窗口后，下次运行需要重新设置。

## 九、如何找到火山方舟推理接入点 ID

大致路径如下：

1. 打开火山方舟控制台。
2. 进入“在线推理”或“推理接入点”页面。
3. 如果还没有接入点，先创建一个推理接入点。
4. 选择你要使用的豆包模型。
5. 创建完成后，复制 Endpoint ID。

Endpoint ID 通常长这样：

```text
ep-xxxxxxxxxxxxxxxx
```

把它填到：

```cmd
set AI_MODEL=ep-xxxxxxxxxxxxxxxx
```

## 十一、使用网页生成报告

打开网页后，按下面步骤操作：

1. 填写“SKU结构”。
2. 填写“进销存表现”。

5. 点击“生成分析报告”。
6. 等待 AI 返回报告。
7. 可以点击“复制报告”或“打印报告”保存结果。

如果只是想快速体验，可以点击：

```text
填入示例数据
```

然后直接点击：

```text
生成分析报告
```

## 十二、一键启动方式

项目里也提供了 Windows 一键启动脚本：

```text

```

可以双击运行它。

不过新手建议先学会 CMD 方式，因为 CMD 方式更容易看到错误原因，也更方便配置 API Key。

如果要用 `start.bat`，需要注意：

## 十六、常见问题

### 1. 页面提示未配置 API Key

说明后端没有读取到 `AI_API_KEY`。

请在启动前确认已经在同一个 CMD 窗口执行：

```cmd
set AI_API_KEY=你的火山方舟APIKey
```

然后再运行：

```cmd
python main.py
```

### 2. 页面上配置了 API Key，但还是失败

当前桌面端项目主要读取后端环境变量。

因此推荐用 CMD 的 `set AI_API_KEY=...` 方式配置，而不是只在网页里填写 API Key。

### 3. 报错模型不存在

常见原因是 `AI_MODEL` 填错。

火山方舟通常需要填推理接入点 ID，格式一般是：

```text
ep-xxxxxxxxxxxxxxxx
```

请不要直接填写普通模型名称，除非你的接入方式明确支持。

### 4. 报错 401 或 Unauthorized

可能原因：

- API Key 填错。
- API Key 多复制了空格。
- API Key 已失效。
- API Key 不属于当前项目空间。

请重新复制 API Key 后再设置：

```cmd
set AI_API_KEY=你的火山方舟APIKey
```

### 5. 报错 403 或 Forbidden

可能原因：

- 当前 API Key 没有权限调用该模型。
- 推理接入点没有开通成功。
- 火山方舟账号或项目权限不足。

请检查火山方舟控制台里的模型权限和推理接入点状态。

### 6. 报错 429 或额度不足

可能原因：

- 账号额度不足。
- 请求太频繁。
- 当前模型有频率限制。

可以稍后重试，或检查火山方舟控制台的额度。

### 7. 端口 8080 被占用

如果提示端口被占用，可以修改启动方式，例如：

```cmd
uvicorn main:app --host 0.0.0.0 --port 8090
```

然后浏览器访问：

```text
http://127.0.0.1:8090
```

### 8. pip 安装失败

可以尝试：

```cmd
python -m pip install --upgrade pip
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 9. 生成报告很慢

可能原因：

- 网络较慢。
- 模型响应较慢。
- 输入内容较长。
- 当前模型排队或限流。

一般等待 30 秒到 2 分钟是正常的。

## 十七、PowerShell 用户怎么运行

如果你用的是 PowerShell，不是 CMD，环境变量写法不同。

PowerShell 中应该这样写：

```powershell
$env:AI_API_KEY="你的火山方舟APIKey"
$env:AI_BASE_URL="https://ark.cn-beijing.volces.com/api/v3"
$env:AI_MODEL="你的推理接入点ID"
python main.py
```

如果你不确定自己用的是 CMD 还是 PowerShell：

- CMD 通常窗口里显示 `C:\Users\...>`。
- PowerShell 通常显示 `PS C:\Users\...>`。
