# 门店商品经营AI分析助手

基于「易物+悦人」双轨共生零售经营逻辑，智能分析门店经营数据并输出可落地的商品经营管理优化方案。

## 功能特性

- **数据输入**：分区块录入SKU结构、进销存表现、品类结构、可选补充数据
- **AI智能分析**：调用大语言模型，自动生成专业经营建议
- **双维度输出**：
  - 📦 易物提销 · 商品价值重构
  - 💫 悦人增黏 · 情绪价值赋能
- **优先级落地动作**：3条最高优先级执行方案及预期效果
- **一键复制/打印**：方便保存和分享报告

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置AI API

**方式一（推荐）**：启动后在页面上点击「配置API」按钮填写

**方式二**：环境变量
```bash
export AI_API_KEY="your-api-key-here"
export AI_BASE_URL="https://ark.cn-beijing.volces.com/api/v3"
export AI_MODEL="doubao-1-5-pro-256k-250115"
```

**方式三**：修改 `main.py` 顶部的配置区

### 3. 启动服务

```bash
python main.py
```

或

```bash
uvicorn main:app --host 0.0.0.0 --port 8080
```

### 4. 访问应用

打开浏览器访问：`http://127.0.0.1:8080`

## 使用流程

1. **录入数据**：在首页表单中填写门店经营数据（至少填写带 * 号的必填项）
2. **快速体验**：点击「填入示例数据」可自动填充模拟数据
3. **生成报告**：点击「生成分析报告」按钮
4. **查看结果**：在结果页查看AI生成的双维度经营建议
5. **复制/打印**：使用工具栏按钮保存报告

## 支持的AI接口

默认使用OpenAI兼容接口，支持：
- 火山方舟（豆包大模型）
- OpenAI API
- 阿里云百炼
- 其他OpenAI兼容接口

## 技术栈

- **后端**：Python + FastAPI + httpx
- **前端**：原生 HTML + CSS + JavaScript
- **AI调用**：OpenAI兼容接口（SSE流式/同步）

## 注意事项

- API Key属于敏感信息，请妥善保管
- 前端配置存储在浏览器本地（localStorage），刷新页面不会丢失
- 建议在本地或内网环境运行，避免API Key泄露
- 生成报告需要联网调用AI服务，请确保网络通畅
