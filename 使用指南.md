# 门店商品经营AI分析助手 - 快速使用指南

## 🚀 最简启动方式（推荐）

### Windows 用户
1. **双击运行** `start.bat` 文件
2. 等待自动安装依赖（首次约1-2分钟）
3. 浏览器自动打开应用页面

### Mac / Linux 用户
1. 打开终端，进入项目文件夹
2. 执行命令：
   ```bash
   chmod +x start.sh
   ./start.sh
   ```
3. 浏览器自动打开应用页面

---

## 📋 使用流程（3步搞定）

### 第1步：配置AI API
进入页面后，点击顶部 **「⚙️ 配置API」** 按钮：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| API Base URL | `https://ark.cn-beijing.volces.com/api/v3` | 火山方舟接口地址 |
| API Key | 空 | **必填**，从火山方舟控制台获取 |
| 模型名称 | `doubao-1-5-pro-256k-250115` | 豆包大模型 |

> 💡 没有API Key？访问 [火山方舟](https://console.volcengine.com/ark/) 注册并创建一个API Key

### 第2步：录入门店数据
填写4个数据区块（带 * 的为必填）：

```
📦 SKU结构        → 总SKU量、鞋/服/配占比、新品/滞销品占比
📊 进销存表现      → 销售占比、动销率、周转天数、售罄率
🏷️ 品类结构       → 销售贡献、价格带、爆款/长尾占比
➕ 可选补充数据    → 会员偏好、场景数据、竞品分析（选填）
```

**快速体验**：点击 **「📋 填入示例数据」** 可自动填充模拟数据

### 第3步：生成报告
点击 **「🚀 生成分析报告」** 按钮，等待AI分析完成后：

```
📋 报告概述
📦 易物提销 · 商品价值重构   ← 商品结构/动销/库存/社交货币化建议
💫 悦人增黏 · 情绪价值赋能   ← 场景组合/IP联名/社群/体验型商品建议
🎯 优先级落地动作            ← 3条最高优先级执行方案+预期效果
```

点击 **「📋 复制报告」** 可将完整内容复制到剪贴板

## ❓ 常见问题

**Q: 页面提示「未配置AI API Key」怎么办？**
> A: 点击页面「配置API」按钮，填入从火山方舟获取的API Key后点击保存

**Q: 生成报告时卡住或报错？**
> A: 检查网络是否正常连接，API Key是否有效，模型名称是否填写正确

**Q: 如何停止程序？**
> A: 在启动的终端窗口中按 `Ctrl+C` 即可停止服务

**Q: 可以换其他AI模型吗？**
> A: 可以，支持任何OpenAI兼容接口（OpenAI、阿里百炼、智谱等），在配置中修改Base URL和模型名即可
